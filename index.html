Mue

A modern open-source, customisable new tab for modern browsers.


Add To

Example of tab

Performance

Mue provides an amazing customisable user experience, while using the least amount of CPU usage as possible! In all tests conducted Mue used less than 2% of the CPU.

Features

Mue gives you a random quote and background image. It also allows you to customise which of these you wish to see, for example if you did not want the greeting you can easily hide it.

About devs

Mue is a project on Github started all the way back in 2017, it was originally created by David R. But since then, many people have joined forces to create Mue as it is today.
