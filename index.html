Why should I use?

Performance

Mue provides an amazing customisable user experience, while only using a very small amount of your system's resources when open! We use the latest technology to keep it as low as possible.

Features

Mue gives you a random quote and background image from our automatically updating API. It also has offline images and a search bar!

Commitment

We are committed to bringing you a great experience. The original idea for Mue began in 2017 and we have been actively working on it ever since.

Privacy

Unlike many, we actually care. The community version of Mue is entirely opensource. Extension, API, you name it. Check out our GitHub organisation at https://github.com/getmue

Want to contribute? Mue (and this website) is opensource!
